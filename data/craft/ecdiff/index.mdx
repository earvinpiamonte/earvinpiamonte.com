Beautifies an Eclipse IDE code patch on a web browser. The new version of previously known as [Eclipse Patch Viewer](/eclipse-patch-viewer) bookmarklet.

## Usage

1. Open a code patch file on a Chrome browser
2. Click on the ecdiff extension icon

## Supported code patch

- Unified diff with Eclipse IDE
- Unified diff with CVS
